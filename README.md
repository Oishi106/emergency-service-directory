## About the Project

Emergency Service Directory is a web platform designed to help users quickly find and access essential emergency services (e.g., hospitals, fire stations, police, ambulance) in their area. The directory offers a simple, intuitive interface that lists detailed contact information, locations, and service types, enabling users to connect with the right help during critical situations.



Why this matters: In emergencies, having a centralized, up-to-date directory of services can save precious time. This project aims to be that trusted, go-to resource for citizens and first responders alike.

# Key Features

## Service Details Page

- Each service has a profile with: name, address, phone number, map location, and service description.

- Easy-to-copy contact details for quick dialing or mapping.


## Responsive Design

- Fully responsive UI: works on mobile, tablet, and desktop.

- Clean, minimal design for fast access during stress.

## Data Management

- Centralized dataset for emergency service records.

- Easy to update: adding, editing, or removing entries. 

## Offline or Bookmark Capability

- Users can bookmark the directory or save key contacts for future reference.

- Potential for offline access or caching for faster load in low connectivity.
